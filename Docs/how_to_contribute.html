코드 기여 (문서화 포함)

  1. HealthTag Github 저장소를 clone하세요.

    git clone git@github.com:Dev-Dream/HealthTag.git

  2. 코드를 수정하고 저장하세요.

  3. 패치 파일을 생성하세요.

    git diff --no-prefix > filename

    파일이름은 다음 규칙을 따라주세요.

    • issueID.author.yymmdd.patch

    ex) 2.rasauq1122.200624.patch

  4. 개발자에게 해당 패치 파일을 전달해주세요. (이메일 등을 통해)
