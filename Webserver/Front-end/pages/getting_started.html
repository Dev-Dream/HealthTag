시작하기

전제 조건

  • MySQL
  • Node.js

다운로드

Git을 통해 소스 코드를 다운로드하실 수 있습니다.

> git clone git@github.com:Dev-Dream/HealthTag.git

구동에 필요한 프로그램을 설치하세요.

> sudo apt-get install mysql-server
sudo apt install nodejs

준비하기

1. 데이터베이스 계정에 대한 비밀번호 설정을 마쳐주세요.

> sudo mysql -uroot USE mysql;
UPDATE user SET plugin="mysql_native_password" where user="root";
ALTER USER "root"@"localhost" IDENTIFIED WITH mysql_native_password BY "mysql";
CREATE DATABASE healthtag;


2. 데이터베이스를 복원하세요.

> mysql -uroot -p healthtag < DataBase/healthtag/healthtag.sql


3. /Webserver/Back-end/src/important/db.js를 다음 코드로 채우세요.

module.exports = { local : { host : "localhost", user : "root", password : "your_mysql_password", // it will use when server accesses DB database : "your_db_name" //recommend healthtag } }


4. /Webserver/Back-end/src/important/session.js를 다음 코드로 채우세요.

session : { secret: "your_secret_string", // whatever it is, it will be okay resave: false, saveUninitialized: true } }


5. /Webserver/Back-end/src/important/password.js를 다음 코드로 채우세요.

module.exports = { password : "your_own_password" // it will use when you access admin page. }

실행하기

1. clone 받은 로컬 경로로 이동해 다음 명령어를 입력하세요.

2. 웹 브라우저를 켜 주소창에 localhost:3000을 입력하세요.
